Must-remember keywords

  • 3400: best-estimate, hypothetical, negative assurance, Clause 3 Sch II
  • 3402: user/service auditor, Type 1/2, CUECs, restricted use
  • 3410: Scope 1/2/3, emission factors, specialist, tCO₂-e
  • 3420: pro-forma adjustment, applicable criteria, responsible party, compilation only

5 Exam Triggers

  1. Start-up 7-yr feasibility with hypothetical assumptions for banker → SAE 3400 projection; assumptions consistent with purpose; negative + positive opinion.
  2. Depreciation per Income-tax Act in PFI vs Companies Act in historical FS, no disclosure → SAE 3400 inconsistent basis → qualified/adverse or withdraw.
  3. User auditor needs assurance on design + operating effectiveness of outsourced payroll SO throughout year → SAE 3402 Type 2; SA 402 link.
  4. SO description relies on CUECs → SAE 3402; report states CUECs not evaluated; objectives achievable only if CUECs work.
  5. Prospectus pro-forma FI; mgmt wants certification of actual post-merger outcome → SAE 3420 only on compilation per criteria; no outcome assurance; Clause 3 Sch II CA Act bars vouching.

Reporting bottom-lines

  • 3400: negative assurance on assumptions + positive on preparation & presentation; caveat achievability; never vouch for accuracy.
  • 3402: positive opinion on description + design (Type 1) or + effectiveness (Type 2); restricted-use; Type 2 attaches tests + deviations.
  • 3410: reasonable/limited assurance on GHG statement (not in PDF).
  • 3420: positive opinion on compilation per applicable criteria; EOM allowed; modify/withdraw if responsible party won't fix.
